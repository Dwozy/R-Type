Prefab JSON Syntax

At the root of the prefab json file, you have to give the prefab a name and its components this way:

{
"name": "name",
"components": [
...
]
}

Then, you can precise the components taken by the prefab this way:

{
"type": "NameOfTheComponent",
"value": {
"field1": 0,
"field2": "foo",
"field3": {
"field4": "bar",
"field5": 5
}
}
}

Here's the list of custom types in json syntax: Input::Keyboard::Key

"KeyName"

Rect<T>

{
"left": T,
"top": T,
"width": T,
"height": T
}

Vector2<T>

{
"x": T,
"y": T
}

Texture

{
"path": string,
"area": Rect<int>
}
